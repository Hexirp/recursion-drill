演習する際は、用意されたファイル中の undefined を変更し、テストして動作を確認して下さい。たとえば、1.hs を書き換えた後は、以下のようにしてテストできます。

    % runghc 1.hs

答えは「数字ans.hs」というファイルにあります。考える前に答えを見てはいけません。

なお、テストに利用している hspec ライブラリは、以下のようにしてインストールして下さい。

    % cabal update
    % cabal install hspec
